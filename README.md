###Git ellenorzese:
git -v

###nodejs szerver ellenorzese
node -v

npm -v

## Git verziokezeles

git init
git add README.md
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Luki005/Portfolio.git
git push -u origin main

## Igy PUSH-old amikor a fenti setup kesz van:
git remote add origin https://github.com/Luki005/Portfolio.git

## Igy PULL-old amikor mas modositotta a fajlokat:
